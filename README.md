# adex-template

Minimal and Default JS Template for Adex

## Features

- Tailwind (Styles)
- Adex (Monolith)

## Usage

```sh
; npx degit barelyhuman/adex-default-template <project-name>
; cd <project-name>
; npm i 
# or yarn install 
# or pnpm install
; npm run dev
```
