# adex-template

Minimal and Default JS Template for Adex

## Features

- Tailwind (Styles)
- Adex (Monolith)
- Dark Mode ([preachjs/themes](https://github.com/preachjs/themes))

## Usage

```sh
; npx degit barelyhuman/adex-default-template <project-name>
; cd <project-name>
; npm i
# or yarn install
# or pnpm install
; npm run dev
```
